This directory contains files used to build Service Chainer App using Launch4J for windows builds.

1. Install Launch4j
2. Run Launch4j
3. Open config.xml
4. Ensure Icon.ico is populated in Launch4j
5. Ensure FAT jar (with dependencies) is selected in Launch4j
6. Ensure jre folder is selected in Launch4j (This is Oracle JDK 8)
7. Build the service chainer app exe
